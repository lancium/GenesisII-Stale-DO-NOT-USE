Genesis II

Genesis II is a modern compute and data grid system designed and implemented by the Virginia Center for Grid Research team at the University of Virginia. Both existing and emerging web services standards form the basis for the Genesis II system. These include:
  • WS-Addressing
  • WS-Security
  • SAML
  • WS-Naming
  • OGSA-ByteIO
  • OGSA-BES
  • RNS
For more information, please visit us at http://vcgr.cs.virginia.edu
